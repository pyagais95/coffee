# coffee

Данная программа имитирует кофейный аппарат, который может выдавать 3 вида кофе(американо, латтэ, капучино),
регулировать количество сахара и сливок, считать стоимость напитка.

Чтобы сделать заказ, необходимо создать новый экземпляр класса Order

например order = new Order();

далее необходимо выбрать вид кофе(1 - американо, 2-латтэ, 3 - капучино), например набирая,

order.pickSort(1)

мы выбираем американо.
Далее мы можем регулировать количество сахара и сливок

order.addSugar() - добавляет сахар
order.lessSugar -уменьшает количество сахара

order.addMilk() - добаляет молоко
order.lessMilk() - уменьшает количество молока

Чтобы получить кофе, за него необходимо заплатить. Аппарат принимает монеты номиналомЖ
1, 3, 5 и 10 сом.

Чтобы добавить денег на счет необходимо мы вызываем метод addCoin(), например

order.addCoin(10);

добавит 10 сом на счет.

мы можем также посмотреть стоимость заказа вызвав метод  showPrice()

order.showPrice()

покажет сколько стоит заказ в данный момент.

Чтобы получить готовый кофе необходимо вызвать метод getCoffee()

order.getCoffee();

Если средств на счету достаточно аппарат выдаст кофе и сдачу, если она есть.
